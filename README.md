# Arcane Hub Frontend

A Steam-inspired web app built with **React + Vite**, featuring:
- Game search and wishlist system  
- Dynamic, responsive layout  
- Authentication system  
- API integration with backend  

## Tech Stack
- React + Vite
- Tailwind CSS
- Express (backend)
- MongoDB

## Features
- Add/remove games from wishlist
- Search and view game details
- Responsive design inspired by Steam

---

This project is part of the **Arcane Hub** full-stack application.
